# Code Style Guide

1.  **Framework:** Astro with Tailwind CSS.
2.  **Formatting:** Use Prettier (configuration in `.prettierrc.mjs`).
3.  **Language:** TypeScript (configuration in `tsconfig.json`).
4.  **Styling:** Use Tailwind CSS utility classes (configuration in `tailwind.config.mjs`). Avoid custom CSS where possible to maintain consistency.
5.  **Component Structure:** Ensure Astro components follow the standard separation: frontmatter scripts and UI templates.
